# 网络云盘系统（面向实习）
基于QT平台搭建可以搜索好友、添加好友、好友聊天、提供文件同步，备份及共享等服务的网络云存储平台。

# 项目技术 
C++、多线程、SQLite3数据库、TCP Socket网络编程、QT（5.14.2）
